Personvernerklæring

Sist oppdatert: 2025-12-19

Denne personvernerklæringen forklarer hvordan Hockey Pong ("vi", "oss") samler inn og bruker informasjon når du bruker vår nettapplikasjon (kontoer, flerspillerspill, chat, venner og kamphistorikk).

Informasjon vi samler inn

  • Kontodata: brukernavn, e-post (hvis du oppgir det), profilinformasjon, profilbilde.
  • Autentisering og sikkerhet: øktcookies/tokens; 2FA-aktiveringsstatus (hvis du bruker 2FA). 2FA-hemmeligheter lagres sikkert.
  • Sosialt og innhold: venneliste, venneforespørsler, chatmeldinger du sender.
  • Spilldata: kamphistorikk og relatert spillstatistikk.
  • Tekniske data: IP-adresse og grunnleggende logger som er nødvendige for å drifte og sikre tjenesten.

Hvordan vi bruker informasjon

  • For å opprette og administrere kontoer og økter.
  • For å tilby spillfunksjoner (lobbyer/matchmaking) og vise kamphistorikk.
  • For å levere chat- og sosiale funksjoner.
  • For å forhindre misbruk, feilsøke problemer og holde tjenesten sikker.

Informasjonskapsler / Lokal lagring

Vi bruker informasjonskapsler (eller lignende lagring) for å holde deg innlogget og for at applikasjonen skal fungere riktig. Du kan slette informasjonskapsler i nettleseren din, men noen funksjoner kan slutte å virke.

Deling av data

Vi selger ikke dine personopplysninger. Vi deler kun data når det er nødvendig for å drifte tjenesten (for eksempel: hosting-/infrastrukturleverandører) eller hvis det kreves ved lov.
Tjenesteleverandører (hvis aktuelt): Hostet på lokal Docker-distribusjon.

Dataoppbevaring

Vi beholder konto-, chat- og kamphistorikkdata så lenge kontoen din er aktiv, eller så lenge det er nødvendig for å drifte tjenesten. Du kan be om sletting som beskrevet nedenfor.
Kontakt for sletting: Kontakt teamet via prosjektrepositoriets issues: https://github.com/PBjr2002/Transcendence.

Dine valg og rettigheter

  • Du kan oppdatere visse profildetaljer inne i appen.
  • Du kan be om kontosletting ved å kontakte oss på: Kontakt teamet via prosjektrepositoriets issues: https://github.com/PBjr2002/Transcendence.
  • Du kan deaktivere informasjonskapsler i nettleseren din (kan ødelegge innlogging/øktfunksjoner).

Sikkerhet

Vi bruker rimelige tiltak for å beskytte informasjon (for eksempel kryptert transport der det er tilgjengelig og tilgangskontroller). Intet system er 100% sikkert.

Barns personvern

Tjenesten er ikke beregnet for barn under 13 år. Hvis du tror et barn har oppgitt data, kontakt oss så håndterer vi det.

Endringer i denne erklæringen

Vi kan oppdatere denne personvernerklæringen. Vi vil endre datoen for "Sist oppdatert" når vi gjør det.

Kontakt

For personvernspørsmål eller slettingsforespørsler, kontakt: Kontakt teamet via prosjektrepositoriets issues: https://github.com/PBjr2002/Transcendence.
